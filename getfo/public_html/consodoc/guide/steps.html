Predefined steps

Preprocess the input

Short name: “pp”.

Parameters:

pp_in: input XML file. Default: $(in_file).

pp_out: output XML file. Default: $(tmpdir)/$(nodir $(pp_in)).

Action:

Resolve all entites and XIncludes and save the result as $(pp_out).
