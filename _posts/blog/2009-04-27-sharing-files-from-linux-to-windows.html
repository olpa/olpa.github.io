Yes, one installs and configures Samba. All I need is to access a big folder in the read-only mode. The quick and dirty solution is:

In smb.conf:

security = share
guest account = nobody

[my-svn]
  comment = SVN files
  path = /path/to/svn
  read only = yes
  public = yes
  guest ok = yes
