The best CAPTCHA ever

It’s an old joke now, but anyway it worth repeating. And I really like this CAPTCHA:

Are you challenged to solve it?

Even if not, you are welcome to read comments on digg.

It’s hard to trace the origin of this CAPTCHA. Most likely, if I remember correctly, it was first published in Novosibirsk State University, Russia.
